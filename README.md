### Installation

1. Get a free API Key at [https://newsapi.org/](https://newsapi.org/)
2. Clone the repo
   ```sh
   git clone git@github.com:DuyKhangTruong/NewsApp.git
   ```
3. Enter your API in `API/NewsAPI`
   ```js
   private let apikey = 'ENTER YOUR API';
   ```
